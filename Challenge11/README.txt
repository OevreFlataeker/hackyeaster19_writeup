When we intercept the web application's traffic we see that the pictures are requested from
http://whale.hacking-lab.com:1111/pics/XXX with XXX being a number from 1 to 98

Also we see that we can re-request these images in every round and thus see the actual picture number.
As all meme pictures are thankfully different in size we can put them into a dictionary with the size being the key and
end up with a list of two indices for every image :-)

We then only have to call "solve" with these indices as arguments and automatically solve the game every time :-)
